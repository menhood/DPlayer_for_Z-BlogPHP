## 使用方式  
```
[dplayer url="http://xxx.com/xxx.mp4" pic="http://xxx.com/xxx.jpg" autoplay="true" danmu="false"/]
```  
直接在文章插入上述标签即可，默认不自动播放，弹幕开启  
更多说明见 Z-Blog 应用中心 https://app.zblogcn.com/?id=1033
